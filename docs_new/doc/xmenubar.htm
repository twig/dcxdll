XMenuBar
XMenuBar is a command implemented to replace the mIRC menubar.

/xmenubar Commands
The /xmenubar command is used to replace the mIRC menubar.
/xmenubar -c
This command changes the mIRC menubar colors.
Syntax:
/xmenubar -c [+COLORFLAGS] [COLOUR]
Example:
/xmenubar -c +b 255
Parameters:
+COLORFLAGS Color flags.
b Background color.
e Event color.
E Event background color.
i Insert mark color.
l Line color.
s Selected text color.
S Selected background color.
t Text color.
m Message color.
M Message background color.
COLOR The color in $rgb() format.
 
Note. This requires /xtreebar -T to be activated.

/xmenubar -f
This command allows you to change the mIRC Treebar font.
Syntax:
/xmenubar -f [+FONTFLAGS] [CHARSET] [SIZE] [FONTNAME]
Example:
/xmenubar -f +i default 20 Verdana
Parameters:
+FONTFLAGS Font style flags.
a Font is anti-aliased.
b Font is bold.
d Default font.
i Font is italic.
s Font is strikedout.
u Font is underlined.
CHARSET Can be any of the following: ansi, baltic, chinesebig, default, easteurope, gb2312, greek, hangul, mac, oem, russian, shiftjis, symbol, turkish, vietnamese.
SIZE Font size in points.
FONTNAME Font name. Ex: Tahoma, Courier New, Arial, ...

/xmenubar -w
This command allows you to change the mIRC Treebar images.
Syntax:
/xmenubar -w [clear|default | [POS] [+FLAGS] [INDEX] [FILE]]
Example:
/xmenubar -w 0 + 1 icons.icl
Parameters:
clear Clears the icon list.
default Reverts the icon list to the default icons.
POS The position to add the new icon. Use 0 to add to the end.
+FLAGS Icon flags
a Uses the icon associated with the given filetype.
g Convert to grayscale icon.
P If GDI+ is enabled, this will use GDI+ to extract the icon.
 
Note. This requires /xtreebar -T to be activated.

/xmenubar -T
This command allows you to enable or disable DCX controlled drawing of the mIRC TreeBar.
Syntax:
/xmenubar -T [0|1]
Example:
/xmenubar -T 1
 
Note.
  • When enabled, the script function $xtreebar_callback(geticons, [TYPE], [TEXT]) will be called for each item. This will allow you to configure the appearance of each item in the TreeBar.
  • The same alias is called when an item is added to the TreeBar. This will allow you to configure the appearance of the new item.
  • The $xtreebar_callback alias should return 2 image indexes, IMAGE SELECTEDIMAGE. (eg. return 2 5)

$xmenubar Properties
The $xmenubar identifier is used to retrieve information about the DCX XMenuBar implementation.
$xmenubar().icon
Returns the current icons being used by the mIRC TreeBar item.
Syntax:
$xmenubar(N).icon
Example:
$xmenubar(3).icon
Parameters:
N The index of the item.
 
Note. If N is 0, this property retrieves the last item in the mIRC Treebar.

$xmenubar().item
Returns the text of the mIRC TreeBar item.
Syntax:
$xmenubar(N).item
Example:
$xmenubar(3).item
Parameters:
N The index of the item.
 
Note.
  • If N is 0, this property retrieves the total number of items in the mIRC Treebar.
  • Returns $null if unsuccessful.
