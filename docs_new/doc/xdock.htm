XDOCK USED FOR DOCKING! DUH!.


/xdock Command
The /xpopup command is used to create/modify/destroy an XPopup menu.
/xdock -Dock
This command lets you sets border width.
Syntax:
/xdock -Dock [UNKNOWN] N HWND POS NAME
Example:
/xdock -Dock [STILL_WIP] 100

/xdock -DockRefresh
This command lets you sets border width.
Syntax:
/xdock -DockRefresh [UNKNOWN] N
Example:
/xdock -DockRefresh [STILL_WIP] 100
