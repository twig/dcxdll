XThumbnail
XThumbnail allows you to create and modify buttons in the taskbar preview window. These can be used to execute scripted commands without activating the mIRC window.

/xthumbnail Command
The /xthumbnail command is used to manipulate buttons in the taskbar preview window.
/xthumbnail -b
This command adds a button.
Syntax:
/xthumbnail -b [ID] [IMAGE] [TOOLTIP]
Example:
/xthumbnail -b 1 4 This is a new button
Parameters:
ID The unique ID that this button will be recognised by. (1-7)
IMAGE The image index to use.
TOOLTIP The tooltip for the trayicon.

/xthumbnail -e
This command applies all changes made and enables or disables buttons.
Syntax:
/xthumbnail -e [1|0]
Example:
/xthumbnail -e 1
Parameters:
STATE 1 == Enable or 0 == disable.

/xthumbnail -r
This command removes a button.
Syntax:
/xthumbnail -r
Example:
/xthumbnail -r
Parameters:
ID The unique ID that this trayicon will be recognised by.
 
Note. Use 0 for INDEX if the file is a single icon file.

/xthumbnail -w
This command adds an icon to the internal list.
Syntax:
/xthumbnail -w [+FLAGS] [INDEX] [FILENAME]
Example:
/xthumbnail -w + 4 mirc.exe
/xthumbnail -w +g 2 mirc.exe
/xthumbnail -w +P 0 image.png
Parameters:
INDEX Icon index in icon archive
+FLAGS Icon flags
a Uses the icon associated with the given file (as shown in Windows Explorer).
Note: File must exist.
f Uses the icon associated with the given filetype.
Note: Filename is the extension (eg. BMP, PNG, AVI, etc).
g Convert to grayscale icon.
P If GDI+ is enabled, this will use GDI+ to extract the icon.
FILENAME Icon archive filename
 
Note.
  • Use 0 for INDEX if the file is a single icon file.
  • Use -1 for INDEX to load all icons in a file.
  • INDEX supports ranges eg: 1-7,12

/xthumbnail -y
This command clears the image list.
Syntax:
/xthumbnail -y
Example:
/xthumbnail -y

/xthumbnail -W
This command adds a bitmap file.
Syntax:
/xthumbnail -W [FILENAME]
Example:
/xthumbnail -W image.bmp
Parameters:
FILENAME Image filename

xThumbnail Signals
XThumbnail signals are always enabled and sent to mIRC when a button is pressed.
Signal:
DCX thumb mIRC ID
Example:
DCX thumb mIRC 1
Parameters:
thumb The event signal identifier.
ID The unique ID which the button was created with. (1-7)
